## 4. Hashtag — 3 set a rotazione (IG)

Regola: 8-12 per post, mai gli stessi due post di fila (l'algoritmo penalizza i blocchi identici). **Niente #aiart / #aigenerated / #midjourney** (romperebbero il gioco percettivo) e niente tag che dichiarano esplicitamente la macchina fotografica (#streetphotography ok no? meglio evitare i più "camera-specific"): si usano tag di *estetica e medium neutro*, tutti veritieri.

**Set A — Fine art / B&W:**
`#fineart #fineartprints #blackandwhite #monochrome #bnw_greatshots #visualart #contemporaryart #artcollector #limitededition #printsforsale`

**Set B — Estetica documentaria:**
`#humanstories #portraiture #visualnarrative #documentaryart #peopleoftheworld #monochromatic #bnwmood #artoftheday #emergingartist`

**Set C — Collezionismo / vendita:**
`#artprint #wallart #fineartcollector #blackandwhiteart #figurativeart #artforsale #interiorart #gicleeprint #editionprint`

+ sempre 1-2 tag geografici dell'opera (es. `#salvador #brasil`).
